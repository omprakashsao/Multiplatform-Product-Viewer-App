## Setup
1. Clone the repository:

 ```bash
   git clone https://github.com/omprakashsao/Multiplatform-Product-Viewer-App.git
```

2. Change Directory:
    ```bash
   cd ProductListApp
```
3. Install dependencies:
 ```bash
  npm install
```

4. Start the Expo server:
 ```bash
 npx expo start
```


Open in Expo Go app (iOS/Android)
